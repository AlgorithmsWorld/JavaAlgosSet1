## Normalization Forms

1. 1NF
2. 2NF
3. 3NF
4. BCNF
5. 4NF
6. 5NF

#### 1NF

Required conditions:

a. All values should be atomic (non-breakable).  
b. There should be a candidate key or a composite key (basically you should be able to uniquely identify all records in the table).
